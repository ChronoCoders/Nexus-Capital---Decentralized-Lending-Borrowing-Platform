## 🚀 Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/chronocoder/nexus-capital.git

# Install dependencies
yarn install

# Run tests
yarn test

# Deploy to local network
yarn deploy:local
```

1. Connect Wallet
2. Choose Asset Pool
3. Deposit or Borrow
